### 🌧 ระบบเว็บพยากรณ์น้ำฝน

เป็นเว็บพยากรณ์น้ำฝนนี้เป็นเครื่องมือสำหรับวิเคราะห์และคาดการณ์ปริมาณน้ำฝน โดยผู้ใช้งานสามารถ **อัปโหลดไฟล์ข้อมูลในรูปแบบ Excel** เข้าสู่ระบบ 
จากนั้นเว็บไซต์จะประมวลผลและแสดงผลการพยากรณ์โดยอัตโนมัติ รองรับการใช้งานได้บน **คอมพิวเตอร์ โน้ตบุ๊ก และโทรศัพท์มือถือ** อย่างไรก็ตาม
ระบบยัง **ไม่รองรับการใช้งานบน iPad**


#### ข้อกำหนดด้านข้อมูล

* ควรใช้ข้อมูลย้อนหลัง **อย่างน้อย 5 ปีขึ้นไป**
* ช่วงข้อมูลที่เหมาะสมที่สุดคือ **5–10 ปี**
* ข้อมูลควรมีคอลัม วันที่,โรงงาน,หมายเขตเขต,ชื่อเขต,ปริมาณฝนรายวัน (ต้องตั้งชื่อตามนี้)

  * ช่วยให้โมเดลมองเห็นรูปแบบฤดูกาลของฝนได้ชัดเจน
  * ข้อมูลไม่มากเกินไปจนกระทบประสิทธิภาพ
  * และยังมีความเป็นปัจจุบันเพียงพอสำหรับการตัดสินใจ

#### ขอบเขตการวิเคราะห์

ระบบนี้ใช้วิธีวิเคราะห์จาก **สถิติข้อมูลปริมาณน้ำฝนตามเขตพื้นที่ของโรงงาน** เท่านั้น
ไม่ได้รวมปัจจัยภายนอก เช่น

* อุณหภูมิ
* ความชื้น
* ปรากฏการณ์สภาพอากาศอื่น ๆ

ดังนั้นผลการพยากรณ์จึงเป็นการคาดการณ์จากแนวโน้มข้อมูลในอดีต เหมาะสำหรับใช้เป็น **ข้อมูลประกอบการวางแผน** ควบคู่กับดุลยพินิจและแหล่งข้อมูลอื่น
